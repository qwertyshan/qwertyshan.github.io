Where is the Jedi?
Web App

Place your Star Wars character anywhere on earth
by typing a location in free text.

Google Maps API with custom geolocation code.

Built with Python, Bootstrap, and the light side of the Force.

See a Live Demo Connect on LinkedIn

Beautiful Frontend

All views are beautifully designed using the Bootstrap CSS framework.

  • Fully gridded layout
  • Elegant handling of application errors and user feedback

Powerful Backend

  • App is written in Python with webapp2 for URI routing.
  • Jinja enabled frontend templating.
  • Google Map APIs for geo-location and map sourcing.

Fork me on Github

Massively Scalable
Compute and Datastore

The application is built on Google App Engine, with features such as auto-scaling, caching, and sophisticated management dashboards.

Data storage is on NDB, a schemaless object datastore providing robust, scalable storage.
