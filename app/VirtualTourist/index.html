Virtual Tourist
iOS App

Travel virtually around the world.
Drop a pin anywhere on the map and view images for that location.

Built with Swift 2 on Xcode 7.

Explore
the World

Drop a pin anywhere on a world map. Tap pin to see a photo gallery of images from that location.

  • Fully persistent with Core Data (using SQLite)
  • Thread safe implementation
  • Advanced gesture controls for map manipulation

View Images
for Any Location

Image galleries are generated with beautiful effects, for any location tapped on the map.

  • Fully implemented networking client for Flickr API
  • JSON parsing and image caching
  • Beautiful collection views and UI animations
