On the Map
iOS App

Post your location and see other users' locations on a map.
This app was made for Udacity students to share locations with each other.

Built with Swift 2 on Xcode 7.

Login with
Udacity

Authentication enabled with Udacity's API, using rock-solid networking code.

  • Login and logout functionality
  • Network error handling for reliability
  • Secure communication over TLS

Integrated with
Facebook

Secure integration with Facebook Login using Facebook SDK for iOS.

Provides a convenient way for users to authenticate using Facebook instead of having to login seperately to Udacity.

Map View

View other users' locations as pins on a map. Refresh the map with new data with the tap of a button.

  • All data synced to a Parse backend via API calls
  • MapKit with pinch, zoom, scroll gestures
  • Links open gracefully in the mobile browser

List View

View the map data as a list in a table. Refresh the list with new data with the tap of a button.

  • All data synced to a Parse backend via API calls
  • Fully delegated Table View Controller
  • Links open gracefully in the mobile browser

Post Your Location

Post your location using a modal view. Enter your location in a text field, and geolocate your coordinates on a map.

  • All data posted to a Parse backend via API calls
  • Programmatic view manipulation using view containers
  • Geolocation with CLGeocoder
